# EVIL EURO WhatsApp Bot

A feature-rich WhatsApp bot with AI, audio, editing, games, group management, download, and personalization tools. Fully customizable with pairing code support, no QR needed.

## Owner
EVIL EURO

## Commands
All AI, Audio, Editing, Games, Group, Download, Personal, and Misc commands included.

## Deployment
1. Clone repo
2. Run `npm install`
3. Start bot: `npm start`
4. Use pairing code to connect (no QR needed)
